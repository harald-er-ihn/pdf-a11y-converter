# 🛠️ PDF A11y Converter (Standalone)

Ein lokales Desktop- und Kommandozeilen-Tool zur KI-gestützten semantischen Rekonstruktion von PDFs (Erstellung von PDF/UA-1 konformen Dokumenten).

## Status
🚧 **In Entwicklung** (Privates Repository).

## Kern-Technologien
- **Marker** (Layout-Erkennung)
- **BLIP** (Vision-Modell für automatische Alt-Texte)
- **WeasyPrint** (PDF-Generierung)
- **VeraPDF** (Lokale Validierung)
